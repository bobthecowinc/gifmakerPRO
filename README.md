# gifmakerPRO

**gifmakerPRO** is a simple Windows tool that converts one or more images into high-quality animated GIFs. It’s designed to be fast, reliable, and compatible with most platforms.

---

## Features

- Converts multiple images to GIFs in one go.  
- High-quality GIFs using FFmpeg with optimized palette generation.  
- Skips missing files and displays clear error messages.  
- Fully standalone batch-based tool.

---

## Requirements

- **Windows 64-bit** (Windows 7 or newer recommended).  
- **FFmpeg** installed and added to your system PATH.

---

## Download

1. Go to the **[Releases](#)** page.  
2. Download the latest version of `gifmakerPRO.exe`.  
3. Place the EXE in the folder with the images you want to convert.

---

## How to Use

1. Double-click `gifmakerPRO.exe`.  
2. Enter the image filenames you want to convert (separated by spaces).  
   - Example: `1.jpg 2.jpg 3.png`  
3. The program will convert each image to a GIF.  
4. Converted GIFs will appear in the same folder.

---

## Notes

- Make sure the filenames you enter exist in the folder. Missing files will be skipped.  
- Only tested on **64-bit Windows** with FFmpeg available in the PATH.
